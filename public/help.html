Help & Information

About Stroken

Stroken provides access to Steam's Luas and Manifests for free. Our goal is to offer a free steam gaming experiences.

Getting Started

  • Finding Games: Use the search bar on the main page to find games by name or App ID (you can find it on SteamDB).
  • Game Information: Each game card displays its name and App ID, I lnow, this is a lot of information.
  • Downloading Files: Click the "Get Files" button on a game card to access the game's files via Google Drive (you will pass through ouo.io first).

Playing Games for Free

Once you've obtained the files (a .ZIP file contains a Lua and Manifests), follow these general steps:

  1. Install Steam Tools: Doownload, install, and setup Steam Tools on your computer, donwload it from here.
  2. Extract the Archive: Use a program like WinRAR, 7-Zip, or simply, File Explorer, to extract the contents of the downloaded file to a folder on your computer.
  3. Open Steam Tools: After installing and setuping Steam Tools, Open it by double-clicking on its shortcut icon on your desktop, a floating icon will appear on you desktop.
  4. Copy to Steam Tools: After opening Steam Tools and extracting the game's .ZIP file to a folder, you must selet all the files in that folder, then drag them to the floating icon (then release the files), if you were opening Steam from the beginning, you are one step far from seeing the game in your Steam games library (if you were not, stop reading, you have finished, now, you can open Steam and see the game in your library), the last step is restarting Steam, you can do it well by right-clicking on the floating icon, then pressing the "Restart Steam" option (the 2nd option).

Troubleshooting Common Issues

  • Game Not Launching:
    • Ensure all files are extracted and coppied to the Steam Tools watermark.
    • Update your graphics drivers.
    • Install missing DirectX or Visual C++ Redistributables.
  • Missing DLL Errors: Search online for the specific DLL file and how to install it, or try reinstalling the game.
  • Low Performance: Check game's minimum system requirements against your computer's specifications.
